# 🌿 Agro Root Story - Digital Portfolio

A premium, high-performance digital portfolio showcasing impact-driven projects in agriculture and coffee technology. Built with **React 19**, **TanStack Start (SSR)**, and **Tailwind CSS v4**.

## 🚀 Quick Start

### Prerequisites
- Node.js 20+
- npm

### Installation
```bash
# Clone the repository
git clone https://github.com/roedyrustam/agro-root-story-main.git

# Install dependencies
npm install
```

### Development
```bash
# Start dev server
npm run dev
```

### Build & Deploy (Cloudflare)
```bash
# Build for production
npm run build

# Deploy to Cloudflare
npx wrangler deploy
```

## 🎨 Design System
- **Typography**: Butler (Display) & Rubik (Sans)
- **Palette**: Earth Tones (Cream, Coffee, Terracotta, Sage)
- **Animations**: Progressive reveal-on-scroll with SPA-navigation stability.

## 🛠️ Tech Stack
- **Framework**: [TanStack Start](https://tanstack.com/router/v1/docs/guide/start/overview)
- **Routing**: TanStack Router (Type-safe)
- **Styling**: Tailwind CSS v4
- **Deployment**: Cloudflare Pages / Workers
